# Set Up a Virtual Environment:
1. Create it: `python3 -m venv venv` or `python -m venv venv`.
2. Activate it: `source venv/bin/activate`.

# Run Your Application:
1. In your terminal (make sure your venv is active)
2. Run: `python app.py` 
3. Or use the Flask command: `flask --app app run` or `flask run`.
